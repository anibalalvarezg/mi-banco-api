# mi-banco-api

https://mi-banco-api.herokuapp.com/ (Heroku)
API: Nodejs + Express
Database: MongoDB
## External libraries 

*bcryptjs

*cors

*dotenv

*expres

*jsonwebtoken

*mongoose

*morgan

*typescript

## Endpoints

*/api/auth/signup
*/api/auth/signin
*/api/auth/profile

*/api/bank/new-recipient
*/api/bank/recipients
*/api/bank/accounts
*/api/bank/transfer
*/api/bank/transfer-history

## Instalation and run server

Run `npm install` or `npm i` for install dependencies.

Run `npm run start` or `npm run dev` for a dev server on port 3000.
